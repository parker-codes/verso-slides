What is Servo?

  • Created by Mozilla Search in 2012
  • Experiement and improve Firefox
  • Rewrite web engine in Rust

What did it achieve?

  • Modularity: many components are individual crates
    • html5ever
    • cssparser
    • url, and more..
  • Parallelism: for style and layout
  • GPU rendering
    • webrender
    • wgpu
    • pathfinder
